# VinceMed — Site Institucional

Site institucional da VinceMed (distribuidora de produtos médicos para hemodinâmica).

## Rodar localmente

```bash
npm install
npm run dev        # abre em http://localhost:5173 com hot reload
```

Qualquer alteração em `index.html`, `styles.css` ou `main.js` recarrega o navegador automaticamente.

## Build e deploy (Vercel)

```bash
npm run build      # gera /dist
```

Deploy: conecte o repositório na Vercel (preset **Vite** é detectado sozinho) ou rode `vercel` na raiz. Nada mais a configurar.

> Leia o **CLAUDE.md** antes de editar — ele explica o projeto, as decisões de design e as regras.
